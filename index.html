Sobre o HTML:

O HTML (HyperText Markup Language) é um conjunto de símbolos e códigos de marcação, que identificam elementos estruturais da sua página de internet, tais como: parágrafos, títulos, listas, entre outros. Foi criada por Tim Berners-Lee desde 1980 e lançada oficialmente no início dos anos 90 e descrevia 18 elementos da linguagem que permitiam o desenvolvimento inicial usando o HTML.

Sobre o CSS

O CSS (Cascading Style Sheets, em português folha de estilos) é uma linguagem de estilos usada para configurar texto, cor, layout da página e aplicar outros diversos estilos. Dessa forma, a linguagem CSS configura a aparência da página e possibilita aos programadores web separarem o estilo de apresentação das informações de uma página web. As especificações CSS são mantidas pelo grupo World Wide Web Consortium (W3C), que também define os padrões para HTML.
