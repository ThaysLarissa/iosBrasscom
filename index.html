Lista de exercícios

Abaixo está a lista em ordem dos exercícios realizados durante a aula prática de HTML, CSS, JavaScript e React

  1. Subindo o primeiro repositório local para o remoto
    • Utilizando a ferramenta git (git init, git add, git remote, git commit e git push)
  2. Trabalhando com os elementos estruturais do HTML
    • Trabalhando com links e imagens (tag: a e img)
    • Trabalhando com tabelas (tag: table, th, tr e td)
    • Trabalhando com formulários (tag: form, label e input)
  3. Trabalhando com os elementos de CSS
    • Tipos de inserção da folha de estilo do CSS (Externo,interno e Inline)
    • Formatação de linhas, bordas, imagens, diferenciar ID de classe e como aplicar no CSS, conhecer e aplicar as medidas absolutas
